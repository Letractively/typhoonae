==============================
TyphoonAE - Typhoon App Engine
==============================

The TyphoonAE project aims at providing a full-featured and productive serving
environment to run `Google App Engine (Python)
<http://code.google.com/appengine/docs/python/overview.html>`_ applications. It
delivers the parts for building your own scalable App Engine while staying
compatible with Google's API.
